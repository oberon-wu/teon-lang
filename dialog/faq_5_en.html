How to Open
Select a player/NPC nearby by clicking or using target-switch button. Click the palyer/NPC protrait and open the interact menu.

Menu
1. Player
Team: Invite this player to team up.
Add: Add this player as friend. After adding, you can see whether he/she is online and have a private chat in [Menu - Social - Friend].
Trade: Trade with this player.
Whisper: Start a private chat with this player.
Report: Report this player to gm.
Profile: Check player's information.

2. NPC
Chat: Start a chat with NPC.
