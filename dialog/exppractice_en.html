EXP Gaining System Info
1.All classes who reached Lv.49 can use EXP Gaining System.
2.Adventurers can buy Elder's Note and Elder's Advanced Note from Joanna. Each note can only be purchased once per day.
3. The purchase limit will be reset at 00:00 on Teon Server Time. 4. The price of Elder's Note is 2000 Gold.
5. The price of Elder's Advanced Note is 100 Gold.
6.Adventurers can redeem Elder's Note Bag and Elder's Advanced Note Bag from Joanna by using Elder’s Note or Elder’s Advanced Note Bag.
7. Elder's Note Bag = Elder’s Note + Elder's Note Bag Permit (Acquiring from Premium Rewards)
8. Elder’s Advanced Note Bag = Elder’s Advanced Note + 150 TC
9.Expired time of Elder's Note Bag and Elder's Advanced Note Bag are 24 hours and will be disappeared when expired, please inform that.
10.Effective time of EXP Gaining when using Elder's Note Bag is 4 hours, you can acquire a certain amount of EXP after 4 hours.
11.Effective time of EXP Gaining when using Elder's Advanced Note Bag is 1 hour, the total amount of EXP you can acquire is the same as using Elder's Note Bag in 4 hours.
12.The EXP acquired from using Elder's Note Bag is related to the character's level. The higher level the character has, the less EXP will gain from the system.
13.The EXP acquired from using Elder's Advanced Note Bag is not related to the character's level. You will always acquire the same amount of EXP from the system.
14.EXP Cube will not affect the amount of EXP acquiring from this system.
15.The EXP Gaining System is only working in Maros Town, leaving the town or logging out your character will pause the effective time count-down.
16.Characters who reach the maximum level will not able to use EXP Gaining System.
