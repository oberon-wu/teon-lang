Teon Pass Season One Categories:
In Teon Pass Season One, the Basic Teon Pass is unlocked as the default option. Adventurers can use TC to unlock the Advanced Teon Pass Season One to acquire more rewards. The basic daily quests and the advanced daily quests are reset at 00:00 on Teon Server Time. (Teon Server Time is available on Teon official Website at www.teonworld.com).

Teon Pass Level:
Adventurers can acquire TP Points and increase the TP level by complete the daily quests and collect the TP rewards.

Teon Pass Level Limitation:
In the current Teon Pass Season One, when the character's TP level reaches the maximum, will be unable to acquire any TP points and increase TP level.
