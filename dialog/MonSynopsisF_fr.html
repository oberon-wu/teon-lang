Vous pouvez les échanger avec les parchemins suivants :

Pour fabriquer une Parchemin de polymorphe enchanté niveau 1：Parchemin polymorphe niv. 1 x 1、Essence de polymorphe x 5

Pour fabriquer une Parchemin de polymorphe enchanté niveau 2：Parchemin polymorphe niv. 2 x 1、Essence de polymorphe x 8

Pour fabriquer une Parchemin de polymorphe enchanté niveau 3：Parchemin polymorphe niv. 3 x 1、Essence de polymorphe x 11

Pour fabriquer une Parchemin de polymorphe enchanté avancé ：Parchemin de déformation avancé x 1、Essence de polymorphe x 15
