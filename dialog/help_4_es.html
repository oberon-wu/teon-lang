El encantamiento de equipo es fundamental para fortalecer a tu personaje. El equipo encantado proporciona ventajas adicionales, como un mayor daño, precisión y resistencia.
1. Materiales de encantamiento: pergaminos de encantamiento de arma o de armadura.
2. Umbral: cuando intentas encantar una pieza de equipo cuya condición está por debajo del umbral, el equipo no se destruirá si fallas.
3. Resultados posibles: fallo; éxito 1; éxito 2.
Fallo: si el intento de encantamiento fracasa, se consume el pergamino de encantamiento. Si la condición del equipo está por encima del umbral, se destruirá.
Éxito 1: el encantamiento funciona pero el nivel de encantamiento no aumenta.
Éxito 1: el encantamiento funciona y el nivel de encantamiento aumenta.
4. Tipo de pergamino: pergamino de encantamiento benditos, pergaminos de encantamiento malditos.
Pergamino de encantamiento bendito: probabilidad de aumentar el nivel de encantamiento de 1 a 3 si tiene éxito.
Pergamino de encantamiento maldito: reduce el nivel de encantamiento en 1 si tiene éxito.
4.1 Equipo encantado (normal y bendecido)
El nivel de encantamiento puede permanecer igual aunque el proceso de encantamiento se haya realizado con éxito si el nivel de encantamiento del objeto es igual o superior a 8.
4.2 Pergamino de encantamiento de arma bendecido
Si se encanta un arma con un nivel de encantamiento de 1 a 3, existe la probabilidad de aumentar el nivel de encantamiento de 1 a 3 si se realiza con éxito.
Si se encanta un arma con un nivel de encantamiento de 4 a 6, existe la probabilidad de aumentar el nivel de encantamiento de 1 a 2 si se realiza con éxito.
No existen efectos adicionales si se encanta un objeto de nivel de encantamiento 7 o superior.
4.2 Pergamino de encantamiento de armadura bendecido
Si se encanta una armadura con un nivel de encantamiento de 1 a 2, existe la probabilidad de aumentar el nivel de encantamiento de 1 a 3 si se realiza con éxito.
Si se encanta una armadura con un nivel de encantamiento de 3 a 5, existe la probabilidad de aumentar el nivel de encantamiento de 1 a 2 si se realiza con éxito.
No existen efectos adicionales si se encanta un objeto de nivel de encantamiento 6 o superior.
5. Efectos de encantamiento
Encantar una armadura puede reducir el valor de tu AC (aumenta la defensa). Algunos encantamientos pueden aumentar la resistencia a la magia.
Encantar un arma puede aumentar el daño de ataque final. Al alcanzar un determinado nivel de encantamiento, las armas proporcionan un efecto adicional.
Encantar un accesorio proporciona PS adicionales, PM u otros efectos.
6. Pergamino de protección
Un pergamino de protección puede impedir la destrucción de un objeto cuando falla el proceso de encantamiento, pero el nivel de encantamiento del objeto pasará a 0.
No se puede usar un pergamino de protección con un objeto que esté dentro del nivel de seguridad de encantamiento.
Existen diferentes tipos de pergaminos: pergamino de protección, pergamino de protección de accesorio y pergamino de protección de arma avanzado.
El pergamino de protección se puede usar tanto en armas como en armaduras.
El pergamino de protección de accesorio solo se puede usar en accesorios.
El pergamino de protección de arma avanzado solo se puede usar en armas con niveles de encantamiento de 12 o más. También limita la reducción nivel de encantamiento hasta en -3 cuando el proceso de encantamiento falla.
