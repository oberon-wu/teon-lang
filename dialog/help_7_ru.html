Превращение
1. С помощью предметов или навыков персонажи могут превратиться в других существ, изменив свою внешность.
2. В случае превращения в чудовище скорость движения персонажа изменится.
3. В случае превращения в чудовище у персонажей одинакового класса будет разная скорость атаки в зависимости от типа оружия.
4. В случае превращения в чудовище у персонажей, использующих оружие одинакового типа, будет разная скорость атаки в зависимости от класса персонажа.
5. Некоторые чудовища не нападают на превращенных персонажей.
6. Превращенный персонаж не может использовать некоторые предметы снаряжения.

Свитки превращения
1. Свитки превращения можно использовать только на самом себе. У вас будет возможность выбрать, в кого превратиться.
2. Есть два типа свитков: обычные и продвинутые.
3. Продвинутые свитки превращения: делают доступными особых существ. Можно использовать начиная с ур. 50.
4. У обычных свитков превращения есть 3 уровня: 1, 2, 3.
5. Требования к уровню персонажа, соответственно: 10, 20 и 30.

Палочки превращения
1. Палочки превращения можно использовать не только на самом игроке, но существо превращения выбирается случайным образом.

Навыки превращения
1. Можно использовать на других целях, существо превращения выбирается случайным образом.
2. Кольцо превращения позволяет заклинателю выбрать существо превращения.

Свитки превращения: доступные существа
1. Продвинутые свитки превращения: рыцарь смерти, Бафомет, Злой хищник, Темный старейшина.
2. Свиток превращения ур. 1: ночная тень, скелет, старший маг, скелет-лучник, копейщик, скелет с топором и шелоб.
3. Свиток превращения ур. 2: гуль, элитный скелет-лучник, элитный скелет с топором, элитный скелет с копьем, орк-боец, оборотень, Элитный скелет маг, элитный скелет и унголиант.
4. Свиток превращения ур. 3: багбер, ликантроп, Орк маг, три-крин и темный эльф.
