Location
Chat window is at the middle of the botton of game interface. Click the icons on right hand side of chat window can extend or hide the chat window.

Chat Channel
Chat window is sorted as: Hidden, samll window and large window. It can be switched by the icon on the right hand side of chat window.

Diverse chatting features
Various stickers, emojis and customizable font colors.

Channel types
General:
Check [General] button to display messages in all channels except [Combat].
Near:
Check [Near] button to display messages in [Near] channel with a [Near] tag.
World:
Check [World] button to display messages in [World] channel with a [World] tag. Messages in [Shout] channel will also display in [World] channel with a [Shout] tag.
PM:
Check [PM] button to display messages in [PM] channel with a [PM] tag.
PM channel is a one on one chat. You need to enter: Name of the one you want to chat with + space + content, to chat in this channel.
Team:
Check [Team] button to display messages in [Team] channel with a [Team] tag.
You need to in a team to chat in this channel.
Clan:
Check [Clan] button to display messages in [Clan] channel with a [Clan] tag.
You need to in a clan to chat in this channel.
System:
Check [System] button to display messages in [System] channel with a [System] tag. For example: Special dropped items during the battle.
Info:
Check [Info] button to display messages in [Info] channel with a [Info] tag. For example: Gain status.
Combat:
Check [Combat] button to display messages in [Combat] channel with a [Combat] tag. For example: Information about battle.

Shortcut
Click the trumpet icon at the left hand side of chat window, open shortcut setting page and you can set your shortcut for chatting.
Up to 20 shortcuts can be set.
