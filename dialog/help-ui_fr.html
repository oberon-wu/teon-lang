PV
Les points de vie, également appelés PV, représentent la vie du personnage. Quand les PV atteignent 0, le personnage meurt.

PM
Les points de mana, également appelés PM, représentent l'énergie de magie du personnage. Leur principale utilisation est l'invocation de sorts.
Expérience
Les points d'expérience ou XP sont les unités utilisées pour gagner des niveaux d'expérience. Ils sont représentés en pourcentage (%).
Niveau
Représente la puissance globale du personnage et sa progression dans l'aventure. Plus le joueur a de l'expérience, meilleur il sera au combat et en magie.

Classe d'armure(CA)
La classe d'armure, ou CA, représente la défense contre les attaques de monstres. Une CA basse est meilleure qu'une plus élevée

Valeur respectueuse
Représente le caractère respectueux, neutre ou chaotique d'un personnage. De -32768 (totalement chaotique) à 32768 (totalement respectueux).

Poids
Le poids total d'objets dans le sac d'un joueur. Représenté en pourcentage(%).

Faim
Les PV et les PM ne se rétabliront pas automatiquement quand le personnage aura faim.

Fenêtre de chat
Elle est située en bas de l'écran. Le joueur peut discuter avec d'autres joueurs dans cette fenêtre.

Horloge
Elle est située dans le coin supérieur droit de l'écran. Elle affiche l'heure qu'il est dans le jeu.

Raccourci
Il est situé sous le bouton d'attaque. Peut être utilisé comme raccourci.(pour les compétences et les objets)
