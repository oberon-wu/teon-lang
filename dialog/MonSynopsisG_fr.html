Vous pouvez les échanger avec les parchemins suivants :

Pour fabriquer une Parchemin de polymorphe enchanté niveau 1 (lié)：Parchemin polymorphe niv. 1(lié) x 1、Essence de polymorphe x 5

Pour fabriquer une Parchemin de polymorphe enchanté niveau 2 (lié)：Parchemin polymorphe niv. 2(lié) x 1、Essence de polymorphe x 8

Pour fabriquer une Parchemin de polymorphe enchanté niveau 3 (lié)：Parchemin polymorphe niv. 3(lié) x 1、Essence de polymorphe x 11

Pour fabriquer une Parchemin de polymorphe enchanté avancé (lié)：Parchemin de déformation avancé(lié) x 1、Essence de polymorphe x 15
