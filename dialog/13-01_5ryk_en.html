Different classes have different class trainers. You can find your own class trainer in following locations when you reach to the required level.

Royal Class Trial:
Level Requirement: 47
Class Trainer Location: Maros Town.

Knight Class Trial:
Level Requirement: 47
Class Trainer Location: Maros Region.

Elf Class Trial:
Level Requirement: 47
Class Trainer Location: Maros Region.

Darkelf Class Trial:
Level Requirement: 47
Class Trainer Location: Maros Region.

Open the mini map in related region so that you can find them more easily.
