Magia elementale elfica
Istruzioni: Gli elfi possono imparare incantesimi elementali elfici da Lia nel Giardino elfico. L’apprendimento di questi incantesimi prevede un costo in elementi.
1. Puoi imparare incantesimi di un solo elemento specifico alla volta. Se vuoi imparare incantesimi di un altro elemento, dovrai disimparare quelli che hai già appreso. Gli incantesimi disimparati non saranno ripristinati anche se decidi di ritornare allo stesso elemento.
2. Elemento di fuoco: ottieni la furia del Dio del fuoco aumentando le tue doti di lotta in mischia.
3. Elemento di terra: ottieni l’energia della terra per proteggerti meglio
4. Elemento di vento: Percepisci il movimento dell’aria aumentando le tue doti di attacco a lungo raggio
5. Elemento d’acqua: controlla il flusso dell’acqua, migliorando le tue abilità di potenziamento
