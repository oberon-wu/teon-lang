在解释影响魔法伤害的各项因素之前，我想先向你说明两个概念：随机伤害与固定伤害。

随机伤害：顾名思义，即这一部分的伤害值并不是固定的，而是在一个范围内取随机值。

固定伤害：即这个数值所增加的伤害值是固定的。例如魔伤和额外魔伤。

系数：即这个数值会以一定的比例影响伤害值。例如，增加1点魔攻并不代表最终造成的伤害增加1点，而是根据原本造成的伤害，按照一定的百分比增加伤害。一般而言，系数类数值会对伤害较大的魔法造成更大的影响。

接下来，我会按照结算的先后次序讲解魔法伤害结算的流程。（注意：该结算结果只适用于一般的攻击魔法。特殊攻击魔法/技能，例如三重矢、重击、横扫千军等特殊技能，不适用于以下计算方式）。

1.魔法基础伤害
每种魔法都有一定的基础伤害。魔法的基础伤害属于随机伤害。每个魔法都有最大伤害值（即随机范围内可能打出的最大值）和最小伤害值（即随机范围内可能打出的最小值）。一般而言，等级越高的魔法，最小伤害值越高，且最小伤害值和最大伤害值的差值越大。

2.魔伤
魔伤会直接增加1.魔法基础伤害。魔伤在计算时属于固定伤害，会直接按照显示数值增加1.魔法基础伤害的最小伤害值和最大伤害值。假设一个魔法会造成15至20点伤害，增加1点魔伤之后，该魔法会造成16至21点伤害。

3.元素抗性
1点元素抗性会转化为0.01点元素防御系数（会在4.魔攻部分计算结果说明用途）。部分怪物会有弱属性，即表示这种怪物对应的元素抗性为 -50（即 元素防御系数 = -0.5）。

4.魔攻
魔攻取值为【 角色智力 + 装备/状态增加的魔攻值 】。
魔攻是一个系数，它按照以下公式影响1.魔法基础伤害和2.魔伤计算之后的结果：
【 (魔法基础伤害 + 魔伤) x (1 + (魔攻 - 12) x 9.375% - 目标的元素防御系数 ) 】
由此可得，当魔攻小于12时，魔法的威力会减弱。

5.连续魔法伤害衰减机制
当同一个目标在两秒间隔之内连续受到同一招魔法（无论是否来自同一个施法者）的攻击超过两次，则目标受到的来自同一招魔法的伤害会大幅降低。如果每次受到同一招魔法的攻击间隔都不超过两秒，则伤害衰减的效果会叠加，最多叠加到衰减2/3的伤害。该衰减只影响到结算魔攻和元素抗性之后的结果。

6.魔法暴击
1至6级的攻击类普通魔法会有10%的几率产生暴击。法师和精灵可通过配置初始智力提升暴击几率。当魔法暴击时，当前的魔法伤害提高至1.5倍。

7.额外魔伤
法师和精灵的初始智力达到一定的点数可以提升额外魔伤。部分装备或者道具也能提高额外魔伤。额外魔伤不受4.魔攻、5.连续魔法伤害衰减机制以及6.魔法暴击影响，属于固定伤害。

8.魔伤减免
目标的魔伤减免数值会在当前已结算魔法伤害的基础上扣除对应的伤害值。属于固定伤害减免。值得注意的是，魔伤减免结算在12.魔法防御之前。

9.结算圣结界
当目标身上存在圣结界状态时，当前已结算的魔法伤害会下降至1/2，但镜反射的魔法伤害不受目标的圣结界状态影响。

10.镜反射
如果受击目标身上有镜反射，则会对施法者造成当前已结算的魔法伤害，且所造成的伤害不会在双方的魔法防御基础上进行结算，镜反射所造成的伤害会受施放镜反射的角色身上的圣结界状态影响，与被反射角色无关。
即：A对B施放攻击魔法，B身上有圣结界和镜反射，那么A受到1/2的反射伤害；假如A对B施放魔法，A身上有圣结界，B身上有镜反射，此时A的圣结界并不会生效，而是全额受到来自B的反射伤害，且B受到的伤害并不在B的魔法防御基础上进行结算。

11.魔法命中
魔法命中也会对攻击魔法造成影响。各职业均可通过配置初始智力提升魔法命中（规则请咨询属性点导师）。具体魔法命中计算方式会在12.魔法防御计算时说明。

12.魔法防御
魔法防御是一个影响魔法伤害的系数。它的阈值为100。
当目标的魔法防御值小于100的时候：【 当前已结算的魔法伤害 x [ 1 - (目标魔法防御 - 施法者魔法命中) / 2 x 1% ] 】;
当目标的魔法防御值大于等于100的时候：【 当前已结算的魔法伤害 x [ 0.6 - (目标魔法防御 - 施法者魔法命中) / 10 x 1% ] 】;
当对你施法的目标是怪物时，魔法防御的结算方式则会改变：每1点魔法防御代表你拥有1%的几率令来自怪物的魔法伤害减半。

13.冻结和石化减免
如果目标处于冻结或石化状态，则受到的魔法伤害固定减少15点（最少受到0点伤害，不会为负数）。
